# 📝 To-Do List — CRUD MVC em PHP + MySQL e jQuery

Aplicação de lista de tarefas desenvolvida em **PHP** seguindo o padrão **MVC**, utilizando **MySQL** para banco de dados.  
O frontend conta com **HTML**, **CSS**, **JavaScript** e **jQuery**, garantindo uma interface dinâmica e responsiva.  
O projeto é um CRUD completo, permitindo **adicionar, editar, excluir e marcar tarefas como concluídas**.

---

## 🚀 Funcionalidades

- ➕ **Adicionar** novas tarefas
- ✏️ **Editar** tarefas existentes
- ❌ **Excluir** tarefas
- ✅ **Marcar/Desmarcar** tarefas como concluídas
- 📅 Registro automático da data de criação
- 💾 Persistência no banco **MySQL**
- ⚡ Interatividade com **jQuery** e **JavaScript**

---

## 🛠 Tecnologias Utilizadas

- **PHP (MVC)** — Backend organizado em camadas
- **MySQL** — Banco de dados relacional
- **jQuery** — Manipulação de DOM e requisições AJAX
- **JavaScript (ES6)** — Interatividade extra no frontend
- **HTML5** — Estrutura semântica
- **CSS3** — Estilização base
